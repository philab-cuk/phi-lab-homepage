## 2025학년도 2학기 (Fall 2025)

### 의료빅데이터분석 (Biomedical Big Data Analysis)
- **Level**: Undergraduate
- **Language / Tools**: R
- **Description**: Bridges programming knowledge with healthcare analytics.
- **Course objectives include**:
  - Understanding medical data ecosystems (EHR, insurance claims, registry data)
  - Statistical methods (regression, survival analysis, hypothesis testing)
  - Medical coding systems (ICD, SNOMED-CT)
  - Ethical research considerations
  - R programming proficiency for medical data analysis
- **Course images**:
  - https://i0.wp.com/philabcuk.org/wp-content/uploads/2025/10/image-2.png
  - https://i0.wp.com/philabcuk.org/wp-content/uploads/2025/10/image-4.png

### 머신러닝 (Machine Learning)
- **Level**: ❓ (not specified — likely undergraduate)
- **Language / Tools**: R
- **Description**: Comprehensive introduction to statistical learning with R applications. Covers supervised learning (regression, classification, tree-based methods) and unsupervised learning.
- **Prerequisites**: Introductory statistics recommended
- **Course image**:
  - https://i0.wp.com/philabcuk.org/wp-content/uploads/2025/10/image.png

### AI를 위한 프로그래밍 (Programming for AI)
- **Level**: Graduate (Medical AI program)
- **Language / Tools**: Python
- **Description**: Python-based course designed for students from healthcare backgrounds with limited programming experience. Covers:
  - Python basics
  - Data analysis tools
  - Machine learning concepts
  - Clinical data processing
  - Medical imaging applications
